## Reddit Post (r/webdev)

**Title:** [Showoff Saturday] I built an offline-first task PWA with Vite + TypeScript

**Body:**
Hey r/webdev,

I built Clawy Tasks — a task management PWA that works completely offline. No backend, no cloud, no signup.

**Stack:**
- Vite (dev server + build)
- TypeScript (type safety)
- Tailwind CSS (styling)
- Zustand (state management)
- localStorage (persistence)

**Features:**
- Task CRUD with priorities
- Project organization
- Due dates with overdue highlighting
- Filtering by project/priority/status
- Data export (JSON/CSV)
- PWA installable

**Why no backend?** Because solopreneurs don't need another cloud service. Their data should stay on their device.
