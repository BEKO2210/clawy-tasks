# Marketing Copy — Clawy Tasks

## Product Hunt Launch

### Tagline
Simple, offline-first task management for solopreneurs who ship.

### Description
Clawy Tasks is a free, privacy-first task manager that works offline. No signup, no cloud lock-in, no feature bloat. Just create tasks, organize by project, set priorities, and ship.

Built for indie hackers and solo founders who need to stay focused without wrestling with complex project management tools.

**Key features:**
- ⚡ Works offline as a PWA
- 📁 Projects with progress tracking
- 🎯 Priority levels + due dates
- 📊 Dashboard with stats
- 📤 Export to JSON/CSV anytime
- 🔒 Your data stays local

**Pro ($5/mo):** Unlimited projects, cloud sync, advanced analytics, priority support.

Hey Indie Hackers,

I built Clawy Tasks over a weekend because every task manager I tried either:
- Required an account (why?)
- Was slow (cloud sync lag)
- Was expensive ($10+/mo for basic features)
- Locked my data (no export)

So I made the opposite:

**Clawy Tasks** — A free, offline-first task PWA.

- Unlimited tasks
